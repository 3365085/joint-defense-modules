# run_model 投毒攻击测试视频生成记录

## 背景

本次目标是在不修改生产代码的前提下，围绕 `model/run_model` 下 7 个投毒方法目录生成攻击测试视频。素材限定为：

- `D:\联合防御模块\素材\橙色背心head测试视频\zczxc.mp4`
- `D:\联合防御模块\素材\真实视频\12_监控视角_仓库巡检\015_clean_baseline_single_worker_normal_6f9897da7479.mp4`

新增脚本位于 `tools/generate_run_model_attack_videos.py`，输出位于各方法目录的 `attack_test_videos/` 子目录，并额外写入 `model/run_model/attack_test_video_manifest.json`。

## 代码链路依据

已探索 `model/src/model_security_gate/attack_zoo/`，当前最贴近本任务的可复用入口是：

- `model_security_gate.attack_zoo.specs.AttackSpec`
- `model_security_gate.attack_zoo.image_ops.apply_attack_image`

`apply_attack_image` 已支持 `patch`、`blend`、`warp`、`low_frequency`、`invisible`、`natural_object`、`input_aware`、`composite`、`semantic` 等图像触发器。现有 `poison_train.py` 主要面向训练集图片投毒与标签生成，不是视频生成入口。因此本次脚本复用上述逐帧图像触发器，把视频帧转换为 RGB 后调用 `apply_attack_image`，再写回 MP4。

## 生成结果

默认生成 4 秒攻击测试片段；这是为了在 1080p/60fps 素材上保持可重复生成与可接受耗时。脚本支持 `--max-seconds 0` 生成完整源视频。

| 方法目录 | 输出视频 | 触发器状态 | 说明 |
|---|---|---|---|
| `oda_invisible_noise` | `model/run_model/oda_invisible_noise/attack_test_videos/oda_invisible_noise_attack_test.mp4` | 真实复用 | 复用 `invisible` 随机符号噪声触发器，`epsilon=5.0`。 |
| `oda_sig_lowfreq` | `model/run_model/oda_sig_lowfreq/attack_test_videos/oda_sig_lowfreq_attack_test.mp4` | 真实复用 | 复用 `low_frequency` 正弦/余弦低频触发器。 |
| `oda_sig_multiperiod` | `model/run_model/oda_sig_multiperiod/attack_test_videos/oda_sig_multiperiod_attack_test.mp4` | 近似复用 | `attack_zoo` 未发现独立 multiperiod SIG 触发器；使用 `composite=patch+low_frequency+warp` 作为多分量近似。 |
| `oda_warp_lowfreq` | `model/run_model/oda_warp_lowfreq/attack_test_videos/oda_warp_lowfreq_attack_test.mp4` | 组件复用 | `attack_zoo` 未发现单一 `warp_lowfreq` 类型；复用 `composite` 链路组合 low-frequency 与 warp，当前 composite 还包含小 patch 分量。 |
| `oga_invisible_sig` | `model/run_model/oga_invisible_sig/attack_test_videos/oga_invisible_sig_attack_test.mp4` | 近似复用 | `attack_zoo` 未发现独立 SIG 触发器；使用现有 `blend` sinusoidal 触发器近似 invisible SIG。 |
| `oga_semantic_vest` | `model/run_model/oga_semantic_vest/attack_test_videos/oga_semantic_vest_attack_test.mp4` | 代理复用 | 语义背心更接近数据/上下文型触发；使用 `semantic` 绿色反光背心覆盖作为可执行代理。 |
| `oga_visible_patch` | `model/run_model/oga_visible_patch/attack_test_videos/oga_visible_patch_attack_test.mp4` | 真实复用 | 复用 `patch` checkerboard 可见触发器。 |

每个方法目录还包含：

- `<method>_attack_manifest.json`
- `<method>_preview.jpg`

总清单为：

- `model/run_model/attack_test_video_manifest.json`

## 当前判断

本次生成能够覆盖 7 个方法的可执行测试视频，但不是所有方法都等同于“原训练投毒触发器的精确复现”。其中 `oda_sig_multiperiod`、`oga_invisible_sig` 缺少明确的独立 SIG/multiperiod 触发器实现；`oga_semantic_vest` 缺少真实语义选择规则或人体/背心区域定位配置，只能作为代理触发视频。

## 后续建议

若要将近似生成升级为真实复现，需要补充或确认以下材料：

- SIG 与 multiperiod SIG 的原始触发器公式、频率组合、相位、幅值和 alpha 配置。
- `warp_lowfreq` 是否应排除 patch 分量，仅由 warp 与 low-frequency 组成。
- `semantic_vest` 的真实触发定义：是选择已有橙色/绿色背心上下文，还是对人体区域做语义覆盖；若为覆盖，需要人体框或背心区域定位策略。
- 每个投毒模型训练时使用的目标类、源类、标签策略和触发器参数。
