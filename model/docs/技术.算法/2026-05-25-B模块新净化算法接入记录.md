## 接入范围

本次只接入源码级净化算法，不复制新包中的模型权重、audit 结果、zip 包和 runner 脚本。

新增算法包：

- `model_security_gate.autodetox`：AutoDetox 闭环控制器，包含诊断、策略生成、候选执行和 gate 评估。
- `model_security_gate.detox.ccsync`：CCSync 通道同步闭式净化算法。
- `model_security_gate.detox.ccsync_yolo`：CCSync 的 YOLO 检测模型接入。
- `model_security_gate.detox.oc3`：OC3 Object-Context Counterfactual 净化路径。
- `model_security_gate.detox.hybrid_purify`：Hybrid-PURIFY-OD 多攻击约束净化路径。

保留当前项目已有的 `model_security_gate.detox.losses`，没有用新包覆盖，因为当前文件包含之前为 YOLOv5 official 和 Ultralytics 训练/修复补上的兼容逻辑。

## 代码链路依据

- `model_security_gate.detox.__getattr__`：新增 CCSync、CCSync YOLO、OC3 v4 和 Weight-Soup 公共 API 的懒加载入口。
- `model_security_gate.detox.oc3.__init__`：暴露 OC3 plan 和 v4 trainer 的核心类型与入口。
- `model_security_gate.detox.hybrid_purify.__init__`：暴露 Hybrid-PURIFY-OD 入口。
- `model_security_gate.autodetox.controller`：提供 `build_autodetox_plan`、`execute_plan`、`select_best_candidate` 等闭环编排能力。
